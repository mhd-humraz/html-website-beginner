HTML stands for HyperText Markup Language. It is the standard language used to create and design web pages.

What is HTML?

It tells the browser how to display content like text, images, videos, and links.

HTML is made up of elements (also called tags),

written in angle brackets like


<html>, <body>,<p>, etc.

Basic HTML Structure

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>
