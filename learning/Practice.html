CSS Selectors Intermediate

Given this HTML:

<div class="container">
  <p class="highlight">First paragraph</p>
  <p>Second paragraph</p>
  <div>
    <p>Nested paragraph</p>
  </div>
</div>

Write CSS to:

  1. Make all paragraphs red
  2. Make only the .highlight paragraph bold
  3. Make the nested paragraph blue
/* All paragraphs */
p {
  color: red;
}

/* Highlight paragraph */
.highlight {
  font-weight: bold;
}

/* Nested paragraph */
.container div p {
  color: blue;
}

JavaScript Function Advanced

Write a JavaScript function that:

  • Takes an array of numbers as input
  • Returns the sum of all even numbers
  • Example: sumEven([1, 2, 3, 4, 5]) should return 6
function sumEven(numbers) {
  return numbers.reduce((sum, num) => {
    return num % 2 === 0 ? sum + num : sum;
  }, 0);
}

// Alternative solution:
function sumEven(numbers) {
  let sum = 0;
  for (let num of numbers) {
    if (num % 2 === 0) sum += num;
  }
  return sum;
}
