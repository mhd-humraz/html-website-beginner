HTML Cheat Sheets

Quick reference guides for all HTML tags and attributes

Download Complete HTML Cheat Sheet (PDF)
Basic Structure
  • <!DOCTYPE html> — Document type declaration.
  • <html> — Root element.
  • <head> — Meta info, links, title.
  • <body> — Visible page content.
Metadata Tags
  • <title> — Page title.
  • <meta> — Metadata (charset, viewport, SEO).
  • <link> — External resource (CSS).
  • <style> — Internal CSS styles.
  • <base> — Base URL for relative links.
Sectioning Tags
  • <header> — Page/section header.
  • <nav> — Navigation links.
  • <main> — Main content.
  • <section> — Thematic grouping.
  • <article> — Independent content.
  • <aside> — Sidebar content.
  • <footer> — Footer content.
Text Content Tags
  • <h1> to <h6> — Headings.
  • <p> — Paragraph.
  • <br> — Line break.
  • <hr> — Horizontal rule.
  • <blockquote> — Quotation block.
  • <pre> — Preformatted text.
Inline Text Tags
  • <a> — Anchor (link).
  • <strong> — Important text.
  • <em> — Emphasized text.
  • <span> — Generic inline container.
  • <code> — Code snippet.
  • <mark> — Highlighted text.
  • <small> — Small print.
Image and Media Tags
  • <img> — Embed image.
  • <audio> — Embed audio.
  • <video> — Embed video.
  • <source> — Media sources.
  • <track> — Captions for media.
  • <iframe> — Embed external content (like YouTube).
List Tags
  • <ul> — Unordered list.
  • <ol> — Ordered list.
  • <li> — List item.
  • <dl> — Description list.
  • <dt> — Term name.
  • <dd> — Term description.
Table Tags
  • <table> — Table container.
  • <tr> — Table row.
  • <td> — Table data cell.
  • <th> — Table header cell.
  • <thead> — Table head group.
  • <tbody> — Table body group.
  • <tfoot> — Table footer group.
  • <caption> — Table caption/title.
Form Tags
  • <form> — Form container.
  • <input> — Input field.
  • <textarea> — Text area.
  • <button> — Clickable button.
  • <select> — Dropdown list.
  • <option> — Dropdown option.
  • <label> — Label for input.
  • <fieldset> — Group form fields.
  • <legend> — Title for a fieldset.
Attributes
  • id — Unique identifier.
  • class — CSS class name(s).
  • style — Inline styles.
  • href — Link target (in <a>).
  • src — Image or media source.
  • alt — Alternative text for images.
  • title — Tooltip text.
  • target — Link behavior (_blank, _self, etc).
  • disabled — Disable input/button.
  • readonly — Read-only input field.
  • required — Make a field mandatory.
  • placeholder — Hint text in input.
  • value — Default value.
  • name — Form field name.
  • action — Form submit URL.
  • method — Form method (GET/POST).
