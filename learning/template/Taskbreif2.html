Task Management App – Project Brief

Overview:

This is a simple task management (To-Do) app that allows users to add, delete, and filter tasks using local storage. It is built using pure JavaScript and DOM manipulation.

Features:

  • Add new tasks
  • Delete individual tasks
  • Filter tasks by text
  • Persist tasks in browser using Local Storage

Technologies Used:

  • HTML5
  • CSS3
  • JavaScript
  • DOM Manipulation
  • Local Storage

Folder Structure:

/task-app
│
├── index.html
├── style.css
└── script.js

Instructions to Use:

  1. Open index.html in a browser to run the app.
  2. Type in a task and click "Add" to create a task.
  3. Use the delete button to remove a task.
  4. Filter tasks using the search field.

Future Enhancements:

  • Add due dates and priority tags
  • Drag and drop task ordering
  • Progress bar or statistics
