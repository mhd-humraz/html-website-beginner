Blog Website – Project Brief

Overview:

This is a full-featured blog website built using the MERN stack (MongoDB, Express, Node.js). It includes user authentication, the ability to create/edit/delete posts, and an admin panel.

Features:

  • User authentication (login/register)
  • CRUD operations for blog posts
  • Admin panel for managing content
  • Responsive UI

Technologies Used:

  • Node.js
  • Express.js
  • MongoDB
  • HTML/CSS

Folder Structure:

/blog-website
│
├── server/
│   ├── index.js
│   ├── routes/
│   ├── models/
│   └── controllers/
│
└── client/
    ├── index.html
    ├── style.css
    └── scripts.js

Instructions to Use:

  1. Install dependencies using npm install
  2. Run the backend server with node index.js
  3. Open the frontend index.html in browser or serve using Express
  4. Register a user and start posting blogs

Future Enhancements:

  • Rich-text editor for posts
  • Comment section
  • Likes and views counter
  • Theme customization
