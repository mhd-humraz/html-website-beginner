Frames & iFrames Demonstration

Using Frames

Note: The <frameset> and <frame> tags are obsolete in HTML5. This example is for demonstration purposes only. For modern web development, use CSS layout techniques or iframes instead.

Frames divide the browser window into multiple sections, each loading a separate HTML document.

Example Frameset Code:

<!DOCTYPE html>
<html>
<frameset cols="25%,75%">
    <frame src="menu.html" name="menu">
    <frame src="content.html" name="content">
</frameset>
</html>

iFrame Basics

An iframe (inline frame) embeds another HTML page within the current page.

Basic iFrame Example:

Code:

<iframe src="iframe-content.html" width="100%" height="300" title="Example iFrame"></iframe>

iFrame with Custom Attributes:

Code:

<iframe src="iframe-content.html" 
        width="100%" 
        height="300" 
        title="Custom iFrame"
        sandbox="allow-same-origin allow-scripts"
        allowfullscreen>
</iframe>

Security Considerations

When using iframes, consider these security aspects:

  • Use the sandbox attribute to restrict iframe capabilities
  • Consider using X-Frame-Options header to prevent clickjacking
  • Be cautious when embedding content from untrusted sources
  • Use loading="lazy" for performance optimization
