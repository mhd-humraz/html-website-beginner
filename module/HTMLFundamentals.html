Document Structure

Every HTML document has a basic structure that forms the foundation of web pages.

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document Title</title>
</head>
<body>
<!-- Page content goes here -->
</body>
</html>

Key Elements:

  • <!DOCTYPE html> - Declares the document type and HTML version
  • <html> - The root element of an HTML page
  • <head> - Contains meta information about the document
  • <body> - Contains the visible page content
Note: Always include the lang attribute in the <html> tag for accessibility and SEO.

Tables

Tables are used to display data in rows and columns:

Name Email Role
John Doe john@example.com Admin
Jane Smith jane@example.com Editor
Bob Johnson bob@example.com Subscriber
<table> <thead> <tr> <th>Name</th> <th>Email</th> <th>Role</th> </tr> </thead> <tbody> <tr> <td>John Doe</td> <td>john@example.com</td> <td>Admin</td> </tr> <tr> <td>Jane Smith</td> jane@example.com</td> <td>Editor</td> </tr> <tr> <td>Bob Johnson</td> <td>bob@example.com</td> <td>Subscriber</td> </tr> </tbody> </table>

Table Accessibility

For better accessibility:

  • Always use <th> for header cells
  • Use scope="col" or scope="row" on <th> elements
  • Consider adding a <caption> to describe the table
Note: Tables should only be used for tabular data, not for page layout.
